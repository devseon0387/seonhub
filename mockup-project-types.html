hype5 ERP — 프로젝트 타입 확장

실제 프로젝트 화면(카드 3열 그리드 + 큰 필터 박스)에 맞춰 타입 배지/필터/위저드 확장 시안

시안

  • A안 — 카드 상단에 타입 배지 추가 (최소 변경, 카드 레이아웃 그대로)
  • B안 — 검색창 옆 "타입 ▾" 드롭다운 필터 추가 + 카드 배지
  • C안 — 카드 내부에 타입별 전용 정보 블록 (에피소드/스프린트/발행 카운트)
  • D안 — 위저드에 "프로젝트 타입" 선택 스텝 추가
  • E안 — 상세 페이지 타입별 메타/하위 단위 동적 전환

제안 구현 순서
  1. 스키마 — `projects.type` enum (`video`/`dev`/`content`) 추가, 기존 데이터 `video`로 마이그레이션
  2. A안 — 카드 상단 타입 배지만 (1~2시간)
  3. D안 — 위저드 2스텝에 타입 선택 삽입
  4. B안 — 드롭다운 타입 필터 추가
  5. C안 — 카드 하단 "작업 현황"을 타입별 맞춤 지표로
  6. E안 — 상세 페이지 타입별 메타/하위 단위 분기 (가장 무거움)
하위 단위(에피소드/스프린트/콘텐츠)는 공통 테이블 `work_items` + `item_kind`로 묶고, UI만 타입별 라벨/필드 분기하는 게 정산·지출 모듈과의 재사용에 유리.
